# HFT Market-Making Backtester

Python implementation of an inventory-aware HFT market-making framework.

Implemented components:
- replay-based backtester,
- order placement and cancellation,
- execution by market crossing,
- Avellaneda–Stoikov market making,
- enhanced alpha-based quoting,
- parameter sweep experiments.

Main files:
- `hft entrance task.ipynb` — research notebook,
- `report.md` — project report.
